Переменные и типы данных

Что такое переменная?

Переменная — это именованная область памяти, в которой хранится значение. В Python переменная создаётся при присвоении ей значения.

Пример объявления переменной:

name = "Алиса"
    age = 25
    is_student = True

Основные типы данных

  • int — целые числа (10, -5, 42)
  • float — числа с плавающей точкой (3.14, -0.5, 2.71)
  • str — строки ("Hello", 'Python')
  • bool — логические значения (True, False)

Python поддерживает автоматическое определение типа переменной: 

x = 10        # int
    y = 3.14      # float
    text = "Привет" # str
    is_happy = True # bool

Функция type()

Чтобы узнать тип переменной, можно использовать встроенную функцию type(): 

print(type(42))       # 
    print(type(3.14))     # 
    print(type("Hello"))  # 
    print(type(True))     #

Изменение типа данных (кастинг)

Python позволяет изменять тип данных с помощью встроенных функций:

  • int(x) — преобразует в целое число
  • float(x) — преобразует в число с плавающей точкой
  • str(x) — преобразует в строку
  • bool(x) — преобразует в логический тип

Пример:

num = "42"
    converted_num = int(num)  # Преобразование строки в число
    print(converted_num + 8)  # 50
