Модули и пакеты

Что такое модули в Python?

Модуль в Python — это файл, содержащий Python-код. Модули помогают организовать код, разделяя его на логические части. Модуль можно импортировать в другой файл с помощью ключевого слова import. 

# Пример импорта модуля
  import math
  print(math.sqrt(16))  # Выводит: 4.0

Создание собственного модуля

Чтобы создать модуль, достаточно создать файл с расширением .py, например, my_module.py, и затем импортировать его в других программах. 

# Файл my_module.py
  def greet(name):
      return f"Привет, {name}!"
  # В другом файле
  import my_module
  print(my_module.greet("Алексей"))  # Выводит: Привет, Алексей!

Стандартные библиотеки Python

Python поставляется с большим количеством встроенных стандартных библиотек. Например, math, datetime, os, которые предоставляют полезные функции для работы с числами, датами и операционной системой. 

import datetime
  now = datetime.datetime.now()
  print(now)  # Выводит текущую дату и время

Работа с пакетами

Пакет в Python — это директория, содержащая модули и специальный файл __init__.py, который позволяет Python считать директорию пакетом. Пакеты могут содержать другие пакеты (подпакеты). 

# Пример структуры пакета:
  # my_package/
  #     __init__.py
  #     module1.py
  #     module2.py
  # Для импорта из пакета:
  import my_package.module1
