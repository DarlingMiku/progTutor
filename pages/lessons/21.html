Основы работы с библиотеками Python

Что такое библиотеки в Python?

Библиотеки в Python — это коллекции готовых функций и классов, которые позволяют расширить функциональность языка и упростить разработку. Библиотеки могут быть встроенными (например, math, os) или сторонними, которые устанавливаются через менеджер пакетов pip.

Установка библиотек

Для установки сторонних библиотек в Python используется инструмент pip. Чтобы установить библиотеку, необходимо выполнить команду: 

pip install <название_библиотеки>

Например, для установки библиотеки requests, которая используется для работы с HTTP-запросами, необходимо выполнить команду: 

pip install requests

Пример использования библиотеки

В следующем примере показано, как использовать библиотеку math для работы с математическими функциями: 

import math
  
  # Вычисление квадратного корня
  sqrt_value = math.sqrt(25)
  print(sqrt_value)  # 5.0

В этом примере мы импортируем библиотеку math и используем функцию sqrt() для вычисления квадратного корня из числа 25.

Работа с библиотеками сторонних разработчиков

Для работы с внешними библиотеками, как правило, требуется сначала установить их, а затем импортировать в ваш проект. Например, для работы с requests библиотекой, которая упрощает работу с HTTP-запросами, можно написать следующий код: 

import requests
  
  response = requests.get('https://api.github.com')
  print(response.json())

Этот код делает GET-запрос к API GitHub и выводит полученные данные в формате JSON.

Работа с виртуальными окружениями

Для управления зависимостями проекта рекомендуется использовать виртуальное окружение. Это позволяет изолировать библиотеки и версии Python для каждого проекта. Чтобы создать виртуальное окружение, используйте команду: 

python -m venv myenv

После создания виртуального окружения его необходимо активировать: 

# На Windows
  myenv\Scripts\activate
  
  # На macOS и Linux
  source myenv/bin/activate

После активации виртуального окружения вы можете устанавливать библиотеки, и они будут доступны только в этом окружении.
