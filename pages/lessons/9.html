Обработка ошибок и исключений

Что такое исключения?

Исключения — это ошибки, которые возникают во время выполнения программы. Они могут быть вызваны неправильным вводом, делением на ноль, доступом к несуществующему элементу и так далее.

Python позволяет перехватывать исключения и обрабатывать их с помощью блока try-except.

Блок try-except

Блок try используется для выполнения кода, который может вызвать исключение. Блок except используется для обработки исключения, если оно возникает. 

try:
      result = 10 / 0  # Деление на ноль
  except ZeroDivisionError:
      print("Ошибка: деление на ноль!")  # Обработка ошибки

Этот код поймает исключение ZeroDivisionError и выведет сообщение об ошибке.

Обработка нескольких исключений

Можно обрабатывать несколько типов исключений с помощью нескольких блоков except. 

try:
      x = int(input("Введите число: "))
      result = 10 / x
  except ZeroDivisionError:
      print("Ошибка: деление на ноль!")
  except ValueError:
      print("Ошибка: введено не число!")  # Обработка ошибки ввода

Блок finally

Блок finally выполняется всегда, независимо от того, возникло исключение или нет. Он используется для выполнения завершающих операций (например, закрытия файлов). 

try:
      file = open("file.txt", "r")
  except FileNotFoundError:
      print("Файл не найден")
  finally:
      print("Операция завершена")  # Этот блок выполнится в любом случае
