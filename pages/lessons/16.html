Работа с JSON

Что такое JSON?

JSON (JavaScript Object Notation) — это легкий формат обмена данными, который используется для хранения и передачи данных между клиентом и сервером. Он легко читаем человеком и удобно обрабатывается программами.

Основные элементы JSON

JSON представляет собой текстовый формат, основанный на представлении данных в виде пар "ключ-значение" (аналогично словарям в Python).

  • Объекты: Набор пар "ключ: значение", заключенный в фигурные скобки ({})
  • Массивы: Упорядоченный набор значений, заключенный в квадратные скобки ([])
  • Ключи: Строки, которые должны быть заключены в двойные кавычки
  • Значения: Могут быть строками, числами, массивами, объектами или булевыми значениями

Пример JSON

Пример простого объекта JSON:

{
    "name": "Alice",
    "age": 25,
    "isStudent": false,
    "courses": ["Python", "JavaScript", "HTML"],
    "address": {
      "city": "New York",
      "zip": "10001"
    }
  }

Работа с JSON в Python

В Python для работы с JSON используется встроенный модуль json.

Чтобы преобразовать Python-объект в JSON-строку, используйте json.dumps(). Для преобразования JSON-строки в Python-объект — json.loads(). 

import json
  
  # Преобразование Python объекта в JSON строку
  person = {
    "name": "Alice",
    "age": 25,
    "isStudent": False,
    "courses": ["Python", "JavaScript", "HTML"]
  }
  json_string = json.dumps(person, indent=4)
  print(json_string)  # Преобразует в строку JSON
  
  # Преобразование JSON строки в Python объект
  json_data = '{"name": "Alice", "age": 25, "isStudent": false, "courses": ["Python", "JavaScript", "HTML"]}'
  python_obj = json.loads(json_data)
  print(python_obj)  # Преобразует в Python словарь

Чтение и запись JSON в файл

Вы можете сохранить JSON данные в файл и затем считать их обратно: 

import json
  
  # Сохранение данных в файл
  data = {
      "name": "Bob",
      "age": 30,
      "isStudent": True
  }
  with open("data.json", "w") as json_file:
      json.dump(data, json_file)
  
  # Чтение данных из файла
  with open("data.json", "r") as json_file:
      loaded_data = json.load(json_file)
      print(loaded_data)
