Операторы в Python

Арифметические операторы

Python поддерживает стандартные арифметические операции:

  • + — сложение (5 + 3 → 8)
  • - — вычитание (10 - 4 → 6)
  • * — умножение (6 * 7 → 42)
  • / — деление (8 / 2 → 4.0)
  • // — целочисленное деление (9 // 2 → 4)
  • % — остаток от деления (10 % 3 → 1)
  • ** — возведение в степень (2 ** 3 → 8)

Операторы сравнения

Операторы сравнения возвращают True или False:

  • == — равно (5 == 5 → True)
  • != — не равно (3 != 2 → True)
  • > — больше (7 > 3 → True)
  • < — меньше (2 < 5 → True)
  • >= — больше или равно (4 >= 4 → True)
  • <= — меньше или равно (6 <= 8 → True)

Логические операторы

Логические операторы используются для объединения условий:

  • and — логическое "и" (True and False → False)
  • or — логическое "или" (True or False → True)
  • not — логическое "не" (not True → False)

Присваивание с операциями

Можно совмещать присваивание и операции:

x = 5
x += 3  # x = x + 3 → 8
x -= 2  # x = x - 2 → 6
x *= 4  # x = x * 4 → 24
x /= 3  # x = x / 3 → 8.0
