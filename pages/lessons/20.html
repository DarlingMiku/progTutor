Основы многозадачности

Что такое многозадачность?

Многозадачность — это способность операционной системы или программного обеспечения выполнять несколько задач одновременно. В Python многозадачность может быть реализована с использованием потоков и процессов.

Типы многозадачности

В Python многозадачность можно реализовать разными способами:

  • Потоки (Threads): Позволяют запускать несколько задач в одном процессе. Это подход, который подходит для задач, которые могут выполняться параллельно, но не требуют сложной изоляции.
  • Процессы (Processes): Запуск отдельных процессов для выполнения задач. Каждый процесс имеет свой собственный адрес памяти, что делает его более изолированным и стабильным.

Потоки в Python

Для работы с потоками в Python используется модуль threading. Потоки полезны, когда вам нужно выполнять несколько задач параллельно в одном процессе, например, при работе с сетевыми запросами или при параллельной обработке данных.

Пример использования потоков:

import threading
  
  def print_numbers():
      for i in range(1, 6):
          print(i)
  
  thread = threading.Thread(target=print_numbers)
  thread.start()

В этом примере создается поток, который выполняет функцию print_numbers(), выводящую числа от 1 до 5.

Процессы в Python

Для работы с процессами в Python используется модуль multiprocessing. Процессы обеспечивают полную изоляцию, что делает их подходящими для задач, которые требуют значительных ресурсов или требуют работы с большими объемами данных.

Пример использования процессов:

import multiprocessing
  
  def print_numbers():
      for i in range(1, 6):
          print(i)
  
  if __name__ == "__main__":
      process = multiprocessing.Process(target=print_numbers)
      process.start()
      process.join()

Этот код создает новый процесс для выполнения функции print_numbers(). Метод join() ожидает завершения процесса перед продолжением работы программы.

Асинхронность в Python

Асинхронность — это другой подход к многозадачности, который позволяет эффективно использовать ресурсы процессора, выполняя операции, которые требуют ожидания (например, сетевые запросы), без блокировки других задач.

Для реализации асинхронных задач в Python используется модуль asyncio.

Пример асинхронного кода:

import asyncio
  
  async def print_numbers():
      for i in range(1, 6):
          print(i)
          await asyncio.sleep(1)  # имитация асинхронной операции
  
  asyncio.run(print_numbers())

Этот код выполняет асинхронную функцию print_numbers(), при этом каждый шаг приостанавливается на 1 секунду, позволяя другим операциям продолжать выполнение.
