Функции и их использование

Определение функции

Функции в Python определяются с помощью ключевого слова def, за которым идет имя функции и список параметров в круглых скобках.

def greet(name):
        print(f"Привет, {name}!")

Вызов функции

Для вызова функции нужно использовать ее имя и передать необходимые аргументы.

greet("Алексей")  # Выводит: Привет, Алексей!

Возвращаемые значения

Функции могут возвращать значения с помощью оператора return.

def add(a, b):
        return a + b
    
    result = add(3, 4)
    print(result)  # Выводит: 7

Параметры по умолчанию

Можно задавать значения параметров по умолчанию, которые будут использованы, если при вызове функции не переданы аргументы для этих параметров.

def greet(name="Гость"):
        print(f"Привет, {name}!")
    
    greet()  # Выводит: Привет, Гость!
    greet("Алексей")  # Выводит: Привет, Алексей!

Аргументы переменной длины

Для передачи переменного количества аргументов используется звездочка * в параметре функции.

def greet(*names):
        for name in names:
            print(f"Привет, {name}!")
    
    greet("Алексей", "Мария", "Петр")
