Разработка веб-приложений с Flask

Что такое Flask?

Flask — это легковесный веб-фреймворк для Python, который используется для создания веб-приложений. Он предоставляет основу для разработки приложений, от простых до сложных, при этом оставаясь гибким и не перегруженным лишними функциями.

Установка Flask

Для установки Flask нужно использовать менеджер пакетов pip: 

pip install flask

После этого вы можете начать разрабатывать веб-приложение с использованием Flask.

Базовая структура Flask-приложения

Вот пример самой простой структуры Flask-приложения: 

from flask import Flask
  
  app = Flask(__name__)
  
  @app.route('/')
  def home():
      return "Привет, мир!"
  
  if __name__ == "__main__":
      app.run(debug=True)

Этот код создаст приложение с одной страницей, которая будет отображать текст "Привет, мир!" при обращении по адресу http://127.0.0.1:5000/.

Маршруты в Flask

Маршруты в Flask используются для связывания URL-адресов с функциями. Например, можно создать маршрут для страницы "О нас": 

@app.route('/about')
  def about():
      return "О нас"

Теперь, при обращении по адресу http://127.0.0.1:5000/about, приложение отобразит текст "О нас".

Использование шаблонов

Flask поддерживает работу с HTML-шаблонами с использованием библиотеки Jinja2. Вот пример шаблона, который выводит список задач: 

from flask import render_template
  
  @app.route('/tasks')
  def tasks():
      task_list = ['Задача 1', 'Задача 2', 'Задача 3']
      return render_template('tasks.html', tasks=task_list)

В шаблоне tasks.html вы можете использовать переменную tasks, которая будет содержать ваш список: 

<ul>
  {% for task in tasks %}
      <li>{{ task }}</li>
  {% endfor %}
  </ul>

Запуск приложения

После написания кода приложения, его можно запустить командой: 

python app.py

Приложение будет доступно по адресу http://127.0.0.1:5000/ в вашем браузере.
