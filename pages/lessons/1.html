Введение в Python

Что такое Python?

Python — это высокоуровневый язык программирования, который используется в веб-разработке, анализе данных, автоматизации, искусственном интеллекте и многом другом.

Основные преимущества:

  • Простой и понятный синтаксис.
  • Кроссплатформенность (Windows, macOS, Linux).
  • Большое сообщество и множество библиотек.

Как установить Python?

  1. Перейдите на официальный сайт python.org.
  2. Скачайте последнюю версию Python для вашей операционной системы.
  3. Запустите установочный файл и следуйте инструкциям.
  4. Поставьте галочку «Add Python to PATH» при установке (важно для работы в командной строке).
  5. Проверьте установку, введя команду в терминале:
    python --version или python3 --version

Как написать и запустить программу?

Самый простой способ — использовать встроенный интерпретатор Python.

  1. Откройте командную строку (Windows) или терминал (macOS/Linux).
  2. Введите python или python3, чтобы запустить интерпретатор.
  3. Напишите команду print("Hello, world!") и нажмите Enter.

Также можно писать код в файле:

  1. Создайте файл program.py с таким кодом: 
    2. print("Привет, мир!")
  2. Запустите программу командой python program.py в терминале.
