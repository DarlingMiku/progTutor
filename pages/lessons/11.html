Работа с файлами

Открытие файла

Для работы с файлами в Python используется встроенная функция open(). Она позволяет открыть файл для чтения, записи или добавления данных.

# Открытие файла для чтения
  file = open("example.txt", "r")
  content = file.read()  # Чтение содержимого файла
  file.close()  # Закрытие файла

Чтение и запись в файл

Для записи данных в файл используется режим "w" или "a" (для добавления в конец файла). Для чтения используется режим "r".

# Запись в файл
  file = open("example.txt", "w")
  file.write("Hello, Python!")  # Запись строки в файл
  file.close()  # Закрытие файла
  
  # Чтение из файла
  file = open("example.txt", "r")
  content = file.read()  # Чтение содержимого файла
  print(content)
  file.close()  # Закрытие файла

Работа с большими файлами

Когда нужно работать с большими файлами, лучше использовать чтение файла построчно, чтобы не загружать весь файл в память сразу.

# Чтение файла построчно
  file = open("example.txt", "r")
  for line in file:
      print(line.strip())  # Вывод каждой строки
  file.close()  # Закрытие файла

Использование контекстного менеджера

Контекстный менеджер with позволяет автоматически закрыть файл после завершения работы с ним, что улучшает безопасность кода.

# Использование with
  with open("example.txt", "r") as file:
      content = file.read()  # Чтение содержимого файла
      print(content)
  # Файл будет автоматически закрыт после выхода из блока with
