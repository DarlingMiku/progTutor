Объектно-ориентированное программирование (ООП)

Что такое ООП?

Объектно-ориентированное программирование (ООП) — это парадигма программирования, основанная на использовании объектов, которые являются экземплярами классов. Основные концепции ООП включают инкапсуляцию, наследование и полиморфизм.

Создание класса и объекта

В Python для создания класса используется ключевое слово class. Объект класса создается путём вызова класса как функции. 

# Пример класса
  class Car:
      def __init__(self, make, model):
          self.make = make
          self.model = model
  
      def info(self):
          return f"Марка: {self.make}, Модель: {self.model}"
  
  # Создание объекта
  my_car = Car("Toyota", "Corolla")
  print(my_car.info())  # Выводит: Марка: Toyota, Модель: Corolla

Конструктор __init__

Метод __init__ в Python используется для инициализации объектов. Он автоматически вызывается при создании нового экземпляра класса. 

# Пример конструктора __init__
  class Person:
      def __init__(self, name, age):
          self.name = name
          self.age = age
  
      def greet(self):
          return f"Привет, меня зовут {self.name} и мне {self.age} лет."
  
  # Создание объекта
  person = Person("Алексей", 25)
  print(person.greet())  # Выводит: Привет, меня зовут Алексей и мне 25 лет.

Наследование в Python

Наследование позволяет создавать новый класс на основе существующего, наследуя его свойства и методы. Это способствует повторному использованию кода и улучшению структуры программы. 

# Пример наследования
  class Animal:
      def speak(self):
          return "Животное издает звук"
  
  class Dog(Animal):
      def speak(self):
          return "Гав!"
  
  # Создание объекта Dog
  dog = Dog()
  print(dog.speak())  # Выводит: Гав!

Полиморфизм

Полиморфизм позволяет объектам разных классов обрабатывать одинаковые сообщения или вызывать одинаковые методы, но с разным поведением. 

# Пример полиморфизма
  class Cat(Animal):
      def speak(self):
          return "Мяу!"
  
  def make_animal_speak(animal):
      print(animal.speak())
  
  # Создание объектов
  cat = Cat()
  dog = Dog()
  
  # Вызов одного метода для разных объектов
  make_animal_speak(cat)  # Выводит: Мяу!
  make_animal_speak(dog)  # Выводит: Гав!
