Работа с базами данных (SQL)

Что такое SQL?

SQL (Structured Query Language) — это язык запросов для работы с реляционными базами данных. Он позволяет создавать, изменять, извлекать и удалять данные из базы данных, а также управлять ее структурой.

Основные команды SQL

SQL включает в себя несколько основных команд для работы с базой данных:

  • SELECT — извлекает данные из базы данных.
  • INSERT — вставляет новые данные.
  • UPDATE — обновляет существующие данные.
  • DELETE — удаляет данные.
  • CREATE — создает таблицы и другие объекты базы данных.

Создание базы данных и таблицы

Для создания базы данных используйте команду CREATE DATABASE, а для создания таблицы — CREATE TABLE: 

CREATE DATABASE my_database;

Создание таблицы:

CREATE TABLE users (
      id INT PRIMARY KEY AUTO_INCREMENT,
      name VARCHAR(100),
      email VARCHAR(100)
  );

Это создаст таблицу users с тремя колонками: id, name и email.

Вставка данных в таблицу

Для добавления данных в таблицу используется команда INSERT INTO: 

INSERT INTO users (name, email)
  VALUES ('Иван Иванов', 'ivan@example.com');

Эта команда добавит новую строку в таблицу users.

Извлечение данных

Для извлечения данных используйте команду SELECT: 

SELECT * FROM users;

Эта команда вернет все данные из таблицы users.

Обновление данных

Для обновления данных используется команда UPDATE: 

UPDATE users
  SET email = 'newemail@example.com'
  WHERE name = 'Иван Иванов';

Эта команда обновит электронную почту пользователя с именем "Иван Иванов".

Удаление данных

Для удаления данных используйте команду DELETE: 

DELETE FROM users
  WHERE name = 'Иван Иванов';

Эта команда удалит строку с данными пользователя "Иван Иванов" из таблицы.
