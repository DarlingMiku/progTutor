Алгоритмы сортировки

Что такое алгоритмы сортировки?

Алгоритмы сортировки — это методы упорядочивания элементов в списке или массиве по определенному критерию, например, по возрастанию или убыванию. В Python для сортировки списков можно использовать встроенную функцию sort() или sorted().

Основные алгоритмы сортировки

Существует множество алгоритмов сортировки, каждый из которых имеет свои особенности по времени выполнения, сложности и области применения. Мы рассмотрим несколько популярных сортировок:

  • Сортировка пузырьком
  • Сортировка выбором
  • Сортировка вставками
  • Быстрая сортировка (QuickSort)
  • Сортировка слиянием (MergeSort)

Сортировка пузырьком

Сортировка пузырьком — это простой, но неэффективный алгоритм, который многократно проходит по списку, сравнивает соседние элементы и меняет их местами, если они находятся в неправильном порядке.

# Пример сортировки пузырьком
  def bubble_sort(arr):
      n = len(arr)
      for i in range(n):
          for j in range(0, n-i-1):
              if arr[j] > arr[j+1]:
                  arr[j], arr[j+1] = arr[j+1], arr[j]
      return arr
  
  arr = [5, 2, 9, 1, 5, 6]
  print(bubble_sort(arr))  # Выводит: [1, 2, 5, 5, 6, 9]

Сортировка выбором

Сортировка выбором — это алгоритм, который делит список на отсортированную и не отсортированную части. На каждом шаге выбирается минимальный элемент из не отсортированной части и добавляется в отсортированную часть.

# Пример сортировки выбором
  def selection_sort(arr):
      n = len(arr)
      for i in range(n):
          min_idx = i
          for j in range(i+1, n):
              if arr[j] < arr[min_idx]:
                  min_idx = j
          arr[i], arr[min_idx] = arr[min_idx], arr[i]
      return arr
  
  arr = [64, 25, 12, 22, 11]
  print(selection_sort(arr))  # Выводит: [11, 12, 22, 25, 64]

Сортировка вставками

Сортировка вставками — это алгоритм, который строит отсортированный список поэтапно, на каждом шаге добавляя новый элемент в отсортированную часть.

# Пример сортировки вставками
  def insertion_sort(arr):
      for i in range(1, len(arr)):
          key = arr[i]
          j = i - 1
          while j >= 0 and key < arr[j]:
              arr[j + 1] = arr[j]
              j -= 1
          arr[j + 1] = key
      return arr
  
  arr = [12, 11, 13, 5, 6]
  print(insertion_sort(arr))  # Выводит: [5, 6, 11, 12, 13]

Быстрая сортировка (QuickSort)

Быстрая сортировка — это один из самых быстрых алгоритмов сортировки. Он работает по принципу "разделяй и властвуй", разделяя список на две части и рекурсивно сортируя их.

# Пример быстрой сортировки
  def quicksort(arr):
      if len(arr) <= 1:
          return arr
      pivot = arr[len(arr) // 2]
      left = [x for x in arr if x < pivot]
      middle = [x for x in arr if x == pivot]
      right = [x for x in arr if x > pivot]
      return quicksort(left) + middle + quicksort(right)
  
  arr = [3, 6, 8, 10, 1, 2, 1]
  print(quicksort(arr))  # Выводит: [1, 1, 2, 3, 6, 8, 10]

Сортировка слиянием (MergeSort)

Сортировка слиянием — это еще один алгоритм "разделяй и властвуй", который делит список на два подсписка, сортирует их рекурсивно и затем объединяет обратно в один отсортированный список.

# Пример сортировки слиянием
  def merge_sort(arr):
      if len(arr) <= 1:
          return arr
      mid = len(arr) // 2
      left = merge_sort(arr[:mid])
      right = merge_sort(arr[mid:])
      return merge(left, right)
  
  def merge(left, right):
      result = []
      i = j = 0
      while i < len(left) and j < len(right):
          if left[i] < right[j]:
              result.append(left[i])
              i += 1
          else:
              result.append(right[j])
              j += 1
      result.extend(left[i:])
      result.extend(right[j:])
      return result
  
  arr = [38, 27, 43, 3, 9, 82, 10]
  print(merge_sort(arr))  # Выводит: [3, 9, 10, 27, 38, 43, 82]
