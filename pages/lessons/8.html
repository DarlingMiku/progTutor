Словари и множества

Словари

Словарь — это неупорядоченная коллекция данных, которая хранит пары "ключ-значение". Ключи должны быть уникальными, а значения могут быть любого типа.

my_dict = {"name": "Алексей", "age": 25, "city": "Москва"}

Для доступа к значению словаря используется его ключ.

print(my_dict["name"])  # Выводит: Алексей

Методы словаря

Словари имеют полезные методы, такие как get(), keys(), values() и items().

print(my_dict.get("age"))  # Выводит: 25
print(my_dict.keys())  # Выводит: dict_keys(['name', 'age', 'city'])

Добавление и удаление элементов в словаре

Элементы можно добавлять с помощью новой пары "ключ-значение". Для удаления используется метод del.

my_dict["email"] = "alexey@example.com"  # Добавляем новый элемент
    print(my_dict)  # Выводит: {'name': 'Алексей', 'age': 25, 'city': 'Москва', 'email': 'alexey@example.com'}
del my_dict["city"]  # Удаляем элемент
    print(my_dict)  # Выводит: {'name': 'Алексей', 'age': 25, 'email': 'alexey@example.com'}

Множества

Множества — это неупорядоченные коллекции уникальных элементов. Множества не содержат дублирующихся значений.

my_set = {1, 2, 3, 4, 5}

Для добавления и удаления элементов в множестве используются методы add() и remove().

my_set.add(6)  # Добавляем элемент
    print(my_set)  # Выводит: {1, 2, 3, 4, 5, 6}
my_set.remove(3)  # Удаляем элемент
    print(my_set)  # Выводит: {1, 2, 4, 5, 6}
