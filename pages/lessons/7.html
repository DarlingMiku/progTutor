Списки и кортежи

Списки

Списки в Python — это изменяемые последовательности. Они могут содержать элементы разных типов. 

my_list = [1, 2, 3, "Python", True]

Для доступа к элементам списка используется индекс (нумерация с 0). 

print(my_list[0])  # Выводит: 1

Методы списков

Списки имеют множество полезных методов, таких как append(), remove(), pop() и другие. 

my_list.append(42)  # Добавляет элемент в конец списка
    print(my_list)  # Выводит: [1, 2, 3, 'Python', True, 42]
my_list.remove("Python")  # Удаляет элемент из списка
    print(my_list)  # Выводит: [1, 2, 3, True, 42]

Кортежи

Кортежи — это неизменяемые последовательности. После создания кортежа его элементы нельзя изменить. 

my_tuple = (1, 2, 3, "Python", True)

Доступ к элементам кортежа происходит так же, как и у списков, но элементы нельзя изменять. 

print(my_tuple[1])  # Выводит: 2

Сравнение списков и кортежей

Основное отличие между списками и кортежами — изменяемость. Списки изменяемы, а кортежи — нет. Кортежи обычно используются для хранения фиксированных данных, которые не должны изменяться.
