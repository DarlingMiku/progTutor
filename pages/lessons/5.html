Циклы в Python

Цикл while

Цикл while выполняет блок кода, пока условие истинно.

x = 0
    while x < 5:
        print(x)
        x += 1

Цикл for

Цикл for используется для перебора элементов в коллекциях (списки, строки, множества и т.д.).

for i in range(5):
        print(i)

Операторы break и continue

Оператор break завершает выполнение цикла, а оператор continue пропускает текущую итерацию цикла и продолжает выполнение с следующей итерации.

for i in range(10):
        if i == 5:
            break
        print(i)
for i in range(5):
        if i == 3:
            continue
        print(i)

Циклы с else

Циклы for и while могут содержать блок else>, который выполняется, если цикл завершился без использования break.

for i in range(5):
        print(i)
    else:
        print("Цикл завершился")
